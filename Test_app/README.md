# Test_app

## Railsを使ってサンプルのアプリを作ってみる

早速アプリを作っていきましょう。  
Railsでアプリケーションを作成するには以下のコマンドを打ちます。  

    $ rails new アプリ名

今回はTest_appという名前のアプリケーションを作成するので以下のようになります。  

    $ rails new Test_app

するとアプリケーションのひな形となるディレクトリやファイルが作成されます。
